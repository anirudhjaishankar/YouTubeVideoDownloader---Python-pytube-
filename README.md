# Youtube Video Downloader
GUI application to download YouTube video using youtube video's URL.

Copy and paste the youtube video's link in the address bar and press **Download** button.

##Run
`python Youtube.py`

##Pre requisites
`pip install tkinter`
`pip install pytube`
